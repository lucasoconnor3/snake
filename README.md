# Snake (Browser Version)

A simple Snake game that runs in your browser using plain HTML, CSS, and JavaScript.

## Requirements

- Python 3 installed (used only to run a local web server)

## Run Locally

1. Open PowerShell in this project folder:
   - `c:\Users\Lucas\Downloads\snake`
2. Start a local server:
   - `python -m http.server 8000`
3. Open this URL in your browser:
   - `http://localhost:8000/index.html`

Keep the PowerShell window running while you play.  
Press `Ctrl + C` in PowerShell to stop the server.

## Controls

- Move: Arrow keys or `W`, `A`, `S`, `D`
- Restart after game over: `Space` or the **Restart** button
